# TIDEWILD Paddle Board Store

A responsive, editorial e-commerce storefront template for an inflatable stand-up paddle board brand. Inspired by modern European outdoor retail, with original branding, layout, product presentation and interactions.

## Features

- Responsive desktop and mobile navigation
- Product category filters
- Accessible quick-view modal
- Demo shopping bag counter
- Newsletter form feedback
- Scroll reveal motion
- SEO-friendly semantic HTML

## Run locally

Open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy with GitHub Pages

In the repository, open **Settings → Pages**, choose **Deploy from a branch**, select the `main` branch and `/ (root)`, then save.

> This is a front-end demo. Product data, checkout and newsletter submission are sample interactions and are not connected to a commerce backend.
